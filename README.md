# kiostix-server

## Fitur 
-buku
-kategori
-penulis

# Buku
GET
```bash
 https://localhost:3000/api/buku
```
POST
```bash
 https://localhost:3000/api/buku
 request: nama_buku, id_penulis, id_kategori
```

PUT
```bash
 https://localhost:3000/api/buku/{id}
```
DELETE
```bash
 https://localhost:3000/api/buku/{id}
```
